# Shelldio

Music Streaming Service for Developers, Written in Python

## Usage

To run the application simply click on the "shelldio.sh" or "shelldio.bat" files located in the main folder. The shell script can be used for Linux or MacOS, while the batch script can be used to run on Windows. The application should run and you can download and listen to all of your favorite music right from the application. Just simply download this repository and store it wherever you want to on your computer. Another tip for this application is to create a shortcut to the either script file located in the main folder. Once in the app, to see the commands available, please type "help" into the prompt on the screen.
