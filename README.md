# Shelldio

Music Streaming Service for Developers, Written in Python

## Usage

To run the application simply click on the "shelldio.sh" file located in the main folder. The application should run and you can download and listen to all of your favorite music right from the application. Just simply download this repository and store wherever you want to on your computer. Another tip for this application is to create a shortcut to the "shelldio.sh" file located in the main folder. Once in the app, to see the commands available, please type "help" into the prompt on the screen.

***Note: To Run a Shell Script on Windows You Need Another Shell Client, Such as Git Bash***

## Setup

To use the application, you must install the following python packages using the PIP manager. The install commands are shown below for each setup and we have provided a setup command file named "setup.sh" in the main folder.

```pip install pygame```
```pip install pytube```
```pip install moviepy```
